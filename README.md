# Clinical Pharmacy Interventions V2

A modern Clinical Pharmacy Intervention Management System built with:

- HTML5
- Bootstrap 5
- Vanilla JavaScript
- Google Apps Script
- Google Sheets

## Features

- Multi-user ready
- Automatic cloud sync
- Dashboard
- Intervention management
- Drug management
- Disease management
- Reports
- Charts
- Audit log
- Excel export
- PDF export
